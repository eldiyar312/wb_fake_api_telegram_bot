# Telegram BOT Wildberries implantation

## Installation

Project requires [Node.js](https://nodejs.org/) LTS or [Docker](https://www.docker.com/)

Install the dependencies and devDependencies and start the server.

```sh
npm install
cd build
npm ci --production
cp .env ./build
node server.js
```


Run Docker container

```sh
dokcer-comopse up
```
